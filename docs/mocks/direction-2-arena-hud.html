Rationale

The site should feel like Quake the second it loads. This direction borrows the language of the game and its broadcast overlays: warm gunmetal surfaces with a faint scanline, an ember accent straight out of the rocket trail, angular cut-corner plates instead of rounded cards, condensed HUD typography (Rajdhani), and nametags that glow the way the current yellow always wanted to. The medal headers keep their full painted bars — now beveled metal gradients — and the placement columns read as tinted “lanes” with edge lines, the most faithful evolution of today’s identity. Toggles become switch plates with LED dots (LAN ONLY / POWER RANKING feel like arming something). The risk is kitsch, and the answer is restraint everywhere color isn’t: near-black desaturated surfaces, one ember, one yellow. Implementation is still a theme pass — the angles are a clip-path in styleOverrides — but it is the boldest of the three and the one visitors will remember.

Implementation notes (MUI mapping)

  • Theme: shape.borderRadius: 0, palette primary = ember #e05a1f, secondary = name yellow; Rajdhani + Orbitron replace the 8-font pile.
  • Cut corners = one clip-path in MuiPaper/MuiButton styleOverrides; scanline + vignette = two body pseudo-elements in the slimmed App.css (true globals).
  • Medal headers stay full-painted (closest to today) — gradients live in the existing gold-header… classes; lane edges via the placement-cell classes.
  • LED toggle plates = restyled FormControlLabel + Checkbox (icon swapped for the LED diamond) — presentational only.
  • Glow is text-shadow on the existing name class; no images, no canvas, nothing the GH-Pages bundle doesn’t already do.
